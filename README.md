# Titanic Survival Prediction 🚢

## 📌 Objective
To predict whether a passenger survived the Titanic disaster using machine learning.

## 📊 Dataset
Kaggle Titanic Dataset

## ⚙️ Steps
- Data Cleaning
- Encoding
- Feature Selection
- Model Training (Logistic Regression)

## 📈 Results
- Accuracy: ~86%
- Females had higher survival rate
- First-class passengers survived more

## 🛠 Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib / Seaborn
